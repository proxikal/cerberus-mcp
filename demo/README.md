# Cerberus Demo Project

This project exists to test the indexing, parsing, and semantic search capabilities of Cerberus.

## Contents
- `auth.py`: Python classes and functions for authentication.
- `database.ts`: TypeScript database abstractions (imports JS helpers).
- `utils/helpers.js`: JavaScript utility functions (ES modules).
- `utils/py_helpers.py`: Python audit hooks used by `auth.py`.
- `.gitignore`: Filtering rules for the scanner.
